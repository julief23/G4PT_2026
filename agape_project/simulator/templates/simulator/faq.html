AGAPE (computAtional G-quadruplex Affinity PrEdiction) is an AI-based platform designed to predict the ability of small molecules to stabilize G-quadruplex (G4) DNA structures. The platform uses molecular descriptors computed from chemical structures and applies trained predictive models to classify compounds as ACTIVE or INACTIVE toward G4 stabilization.

AGAPE supports three input methods:
  • CSV upload — upload up to 150 molecules in a single file.
  • Single SMILES input — paste a SMILES string and preview the molecule before prediction.
  • Molecule editor (JSME) — draw a structure directly in the browser.

AGAPE currently provides two predictive models:
  • Deep Neural Network (DNN) — uses a compact set of 30 molecular descriptors for fast predictions.
  • Machine Learning Model (XGBoost) — uses 87 descriptors and may provide improved predictive performance.

When users provide molecular structures (SMILES or drawn molecules), AGAPE automatically computes classical molecular descriptors using the Mordred library. These descriptors are then aligned with the feature set required by the selected predictive model.

The AGAPE models were trained on a curated dataset of approximately 1140 molecules with experimentally measured G-quadruplex stabilization data. A large initial pool of molecular descriptors was computed and processed through feature selection techniques to build the final predictive models.

Uploaded molecules or descriptor files are processed only for generating prediction results. They are not permanently stored by the platform. For additional information, please refer to the Privacy Notice.
